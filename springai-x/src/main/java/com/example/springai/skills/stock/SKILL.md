# Stock Skill

## 触发条件

当用户询问以下问题时自动触发：
- "苹果的股价是多少？"
- "特斯拉今天涨了多少？"
- "我的股票账户亏了吗？"
- "微软的PE是多少？"

## 输入参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| symbol | string | 是 | 股票代码（美股如 AAPL、TSLA） |

## 输出格式

```json
{
  "symbol": "股票代码",
  "price": 150.25,
  "change": 2.50,
  "changePercent": 1.69,
  "summary": "简短的股票分析"
}
```

## 执行步骤

1. 接收股票代码参数
2. 调用股票 API 获取实时数据
3. 计算涨跌幅度
4. 生成简短分析
5. 返回格式化结果

## 示例

**输入**: `symbol="AAPL"`

**输出**:
```
Symbol: AAPL
Price: $150.25
Change: +$2.50 (+1.69%)
Summary: AAPL is showing positive momentum today
```

## 注意事项

- 美股代码通常为大写字母
- 中国A股需要使用特殊代码（如 600519.SH）
- 返回的价格单位为美元
